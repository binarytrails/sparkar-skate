# skate

![logo](./demo/logo.png)

> Spark AR Skateboard

Here are all the files used to create an Augmented Reality Skateboard with an early version of Spark AR.

Controls:

    pan         - move
    pitch       - scale
    long press  - rotate
    1x tap      - ollie
    2x tap      - shovit
    3x tap      - kickflip
    4x tap      - ??extra

Demo: https://vimeo.com/509891396
